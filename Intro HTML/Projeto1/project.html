Introdução

HTML (Hypertext Markup Language) é uma ferramenta de marcação de texto, usada para estruturar e mostrar aos usuários de forma organizada páginas e aplicações web. Ao contrário do que se parece, ela não é considerada uma linguagem de programação, mas sim uma linguagem de marcação, sendo entretanto, crucial no desenvolvimento e construção de websites modernos.

Uma das principais vantagens do HTML é a sua universalidade entre navegadores, que independe dos Sistemas Operacionais ou dispositivos dos usuários. Tornando a linguagem consistente e acessível em todos os lugares, e sendo, portanto, de necessidade para qualquer programador front-end.

O HTML é uma linguagem simples e composta principalmente por tags, feitas com o uso do '<' e '>', que são interpretadas pelos navegadores modernos e assim é demonstrado da forma desejada o conteúdo escrito. Essas tags também podem ter atributos próprios de variadas funções. Todo documento HTML começa com o elemento 'html', que é logo em seguida dividido em duas principais partes: 'head', onde se tem informações importantes pre-carregadas da página, que não serão mostradas diretamente ao usuário (código em JavaScript e arquivos de CSS), e 'body', parte que o usuário vê diretamente.


Algumas das principais tags usadas no desenvolvimento de páginas web em HTML: voltar

  • title - Altera o título da página na parte superior do navegador
  • i - Muda o texto para itálico
  • strong - Muda o texto para negrito
  • input - Cria um campo para se receber alguma informação a ser digitada pelo usuário
  • img - Adiciona uma imagem a página web
  • h1 até h6 - Serve de título, subtítulo a ser mostrado no texto
  • p - É o corpo do texto, cria paragráfos
  • u - Coloca o texto com underline
  • mark - Mostra o texto com marca texto
  • sup - Coloca um texto na parte superior de uma linha
  • small - Mostra o texto em uma fonte pequena
  • ol - Separa o texto em uma lista ordenada em cada elemento 'li'
  • ul - Separa o texto em uma lista não ordenada (separada por pontos, como a que lê agora)
  • li - Um texto que é um item de uma lista, ordenada ou não
  • a - É um texto que será clicavél e levará o usuário a outro site ou outro local do mesmo site
  • font - Usada para definir a fonte do texto atual, porém já está obsoleta desde o HTML5
  • del - Indica que uma parte do texto foi removida
  • abbr - é usada para especificar uma abreviação ou sigla no texto
  • blockquote - É usado para mostrar o texto como uma
    citação

Atributos que podem ser usados nas tags: voltar

  • id = "" - Cria um identificador que pode ser usado pelo programador para se referenciar algum elemento
  • style = "" - Serve para a formatação em css
  • class = "" - Serve para formatação abstrata em css
  • type = "" -Usado para dizer o tipo de 'input' a ser inserido pelo usuário ou como pop-up que aparecerá ao se passar o mouse em cima de um texto clicavel
  • src = "" - É usada para se dizer a origem (o link) de uma tag (por exemplo uma imagem)
  • width = "" - É usada para mudar o tamanho de algum elemento
  • href = "" - É usada como link clicável para o usuário, em conjunto com o atributo 'target = ""' e com a tag 'a'
  • target = "" - Muda a localização em que o link será aberto (na mesma aba ou em outra aba)
